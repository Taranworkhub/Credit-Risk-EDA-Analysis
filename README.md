# Credit EDA - Data Science Project

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) of a credit
application dataset. The objective is to understand customer
characteristics, identify data quality issues, handle missing values,
and discover patterns that can help in credit risk analysis.

## Dataset

The project uses a credit application dataset containing customer
information such as: - Demographic details - Income and employment
information - Credit-related attributes - Loan application details

## Project Workflow

### 1. Data Loading

-   Imported required Python libraries
-   Loaded the application dataset using Pandas

### 2. Data Inspection

Performed: - Dataset shape analysis - Data type inspection - Statistical
summary - Missing value analysis

### 3. Data Cleaning

Handled data quality issues by: - Removing columns with a high
percentage of missing values - Treating categorical missing values -
Imputing numerical missing values using appropriate statistical methods

### 4. Exploratory Data Analysis

Analyzed relationships and distributions using: - Pandas - Matplotlib -
Seaborn

## How to Run the Project

1.  Clone the repository:

``` bash
git clone <repository-url>
```

2.  Install required libraries:

``` bash
pip install pandas numpy matplotlib seaborn jupyter
```

3.  Open the notebook:

``` bash
jupyter notebook
```

4.  Run: `CREDIT EDA - PROJECT OF DATA SCIENCE.ipynb`
